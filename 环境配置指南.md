# 环境配置指南

## 问题分析

当前遇到的CMake错误主要是因为：
1. **Qt6找不到** - 系统中没有找到Qt6安装
2. **编译器配置问题** - MSYS2编译器与CMake的兼容性问题

## 解决方案

### 方案1：安装Qt6（推荐）

#### 1.1 下载Qt6安装包
- 访问：https://www.qt.io/download-qt-installer
- 下载开源版Qt Online Installer
- 选择Qt6版本，推荐 Qt 6.5.x 或 6.6.x

#### 1.2 安装组件选择
```
必选组件：
- Qt 6.x.x (MinGW 11.2.0 64-bit)
- Qt Creator 6.x.x

可选组件：
- Qt Charts
- Qt Quick
- Qt OpenGL
```

#### 1.3 设置环境变量
```bash
# 添加到系统PATH
C:\Qt\6.x.x\mingw_64\bin
C:\Qt\Tools\mingw1120_64\bin
C:\Qt\Tools\QtCreator\bin
```

### 方案2：使用Qt5（替代方案）

如果无法安装Qt6，可以修改项目使用Qt5：

#### 2.1 下载Qt5
- 访问：https://download.qt.io/archive/qt/
- 下载Qt 5.15.x版本

#### 2.2 修改CMakeLists.txt
项目已经支持Qt5，只需确保有Qt5环境即可。

### 方案3：配置MSYS2环境

#### 3.1 安装Qt6（MSYS2）
```bash
# 在MSYS2终端中执行
pacman -S mingw-w64-x86_64-qt6-base
pacman -S mingw-w64-x86_64-qt6-serialport
pacman -S mingw-w64-x86_64-qt6-network
pacman -S mingw-w64-x86_64-qt6-charts
```

#### 3.2 设置CMake路径
```bash
# 在项目根目录执行
export CMAKE_PREFIX_PATH=/ucrt64/lib/qt6
cmake .. -G "MSYS Makefiles"
```

### 方案4：使用Visual Studio编译器

#### 4.1 安装Visual Studio
- 安装Visual Studio 2022 Community
- 选择"使用C++的桌面开发"

#### 4.2 安装Qt VS Tools
- 在Visual Studio扩展管理器中搜索"Qt VS Tools"
- 安装并配置Qt路径

## 构建步骤

### 完整构建流程

```bash
# 1. 清理之前的构建
cd hostcomputer
rm -rf build

# 2. 创建构建目录
mkdir build
cd build

# 3. 配置CMake（根据环境选择）
# MinGW环境
cmake .. -G "MinGW Makefiles"

# 或者Visual Studio环境
cmake .. -G "Visual Studio 17 2022" -A x64

# 4. 编译项目
# MinGW
mingw32-make

# Visual Studio
cmake --build . --config Release

# 5. 运行程序
./hostcomputer.exe  # Linux/MSYS2
# 或
hostcomputer.exe     # Windows
```

## 常见问题解决

### 问题1：CMAKE_CXX_COMPILER未设置
```bash
# 手动指定编译器
cmake .. -G "MinGW Makefiles" -DCMAKE_CXX_COMPILER=g++
```

### 问题2：Qt模块找不到
```bash
# 手动指定Qt路径
cmake .. -DCMAKE_PREFIX_PATH=C:\Qt\6.5.3\mingw_64
```

### 问题3：SerialPort模块缺失
```bash
# 确保安装了Qt SerialPort模块
# 或者在CMakeLists.txt中移除对SerialPort的依赖
```

## 推荐配置

### 最小环境要求
- **Qt版本**：Qt 5.15.x 或 Qt 6.5.x+
- **编译器**：MinGW 11.2.0 或 MSVC 2022
- **CMake**：3.16+
- **Python**：3.8+ (可选，用于某些构建脚本)

### 推荐开发环境
- **IDE**：Qt Creator 6.x
- **编译器**：MinGW 11.2.0 64-bit
- **Qt版本**：Qt 6.5.3
- **调试器**：GDB

## 验证安装

### 检查Qt安装
```bash
# 检查qmake
qmake -version

# 检查Qt模块
qmake -query QT_INSTALL_PLUGINS
```

### 检查编译器
```bash
# 检查g++
g++ --version

# 检查make
mingw32-make --version
```

### 验证项目构建
```bash
# 在项目根目录
cmake --version
mkdir build && cd build
cmake ..
# 应该能看到成功的配置输出
```
